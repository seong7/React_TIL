# Vanilla JS 에서 Redux 사용하기

## 환경

```bash
$ yarn global add parcel-bundler
또는
$ npm install -g parcel-bundler

프로젝트 폴더에서
$ yarn init -y

parcel 로 dev server 열기 (http://localhost:1234/)
$ parcel index.html

redux 설치
$ yarn add redux
```
